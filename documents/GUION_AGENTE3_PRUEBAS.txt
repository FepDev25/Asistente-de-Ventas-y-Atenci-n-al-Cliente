   - Calcula total: $110 x cantidad

CRITERIOS DE EVALUACIÓN
==============================================

✅ CORRECTO - Agente debe:
- Identificar productos por ID correcto (ADIDAS-001, etc.)
- Recomendar según uso específico (running/casual/montaña)
- Comparar características técnicas precisas
- Usar precio correcto de BD ($110, $180, etc.)
- Verificar stock antes de ofrecer
- Transferir correctamente a Checkout Agent
- Usar persuasión ética (beneficios reales, no inventar)

❌ INCORRECTO - Agente NO debe:
- Inventar productos no existentes en BD
- Ofrecer precios incorrectos
- Recomendar Samba OG para running (es casual)
- Confundir IDs de productos
- Prometer stock sin verificar
- Usar persuasión manipuladora o falsa
